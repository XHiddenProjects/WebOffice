# WebOffice
WebOffice is an all-in-one adminstrative/network/server tool including high security a controls and manages all of your connected devices.

***

## Supported Devices
> Due to having command lines, it only works on desktops.
* Windows
* MAC
* Linux

***

## Required Libraries

**Python (>13.3)**
* PIP
* PSUTIL

**PHP (>8.2)**
* PDO
* mySQL

**Softwares**
* MS Powershell (>7.5.2)
